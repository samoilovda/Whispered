# Whispered (Whisper Fedora) — Продукт для автоматизации транскрипции и генерации контента

## Обзор продукта
**Whispered** — это современное десктопное приложение, предназначенное для преобразования аудио- и видеозаписей (например, звонков в Zoom, интервью, подкастов) в структурированный текстовый контент. Продукт объединяет в себе передовые технологии распознавания речи, идентификации спикеров и возможности локальных языковых моделей (LLM) для автоматического создания статей, постов для соцсетей и аналитических сводок.

## Основные цели
1.  **Экономия времени**: Автоматизация процесса превращения многочасовых записей в готовые к публикации материалы.
2.  **Конфиденциальность**: Использование локальных инструментов (whisper.cpp, LM Studio) позволяет обрабатывать данные полностью офлайн, не передавая их на сторонние серверы.
3.  **Многоформатность**: Генерация контента в различных стилях — от кратких сводок до полноценных лонгридов и постов для соцсетей.

## Ключевые возможности и функции
*   **Высокопроизводительная транскрипция**: Использование `whisper.cpp` с поддержкой аппаратного ускорения (Metal на macOS) для максимально быстрого и точного распознавания речи.
*   **Диаризация (Разделение по ролям)**: Идентификация различных спикеров с помощью библиотеки `pyannote.audio`, что позволяет подписывать, кто именно произнес ту или иную фразу.
*   **Интеллектуальная генерация контента**: Интеграция с локальными LLM (через LM Studio) для:
    *   Извлечения ключевых тем и идей.
    *   Генерации черновиков блогов в форматах: Blog Post, FAQ, Listicle, Summary.
    *   Создания коротких сниппетов для социальных сетей.
*   **Пакетная обработка**: Возможность загрузки и обработки нескольких файлов одновременно.
*   **Современный интерфейс**: Десктопное приложение на базе PyQt6 с поддержкой темной темы и интуитивно понятным управлением.
*   **Экспорт**: Сохранение результатов в форматах Markdown и HTML.

## Техническая реализация
*   **Язык программирования**: Python 3.
*   **Графический интерфейс**: PyQt6 + qdarktheme.
*   **Обработка аудио**: ffmpeg для извлечения звука и конвертации в формат, оптимизированный для Whisper (16kHz WAV).
*   **Движок распознавания**: whisper.cpp (CLI версия) для эффективного использования ресурсов CPU/GPU.
*   **Спикеры**: pyannote-audio (требует Hugging Face token и предобученные модели).
*   **AI Backend**: Локальный сервер LM Studio, работающий через OpenAI-совместимый API (порт 1234).
*   **Архитектура**: Модульная структура (transcriber, diarizer, article_generator, ui), позволяющая легко расширять функционал.

## Способы использования
Приложение может использоваться как через графический интерфейс (`main.py`), так и через консольные скрипты автоматизации (например, `zoom_to_blog.py`), которые позволяют за один запуск пройти весь путь от видеофайла до готового поста в блог.
